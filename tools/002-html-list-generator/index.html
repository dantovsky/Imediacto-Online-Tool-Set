Converting Text to HTML List

The HTML List Generator is an HTML converter online tool that converts text to html list and others text list formats. Using this tool will help you convert some text list and transform it to a unordered or ordered HTML list, as well these text list formats: text separated by lines, spaces, commas, semicolons and some other custom symbols you want to use.

Notes:
The output text list options are generated with a space after the separator symbol. You can use the custom option to generate that output only with symbols without spaces.

Example: type comma (,) on custom option and will be generated a list like (banana,apple,blueberry).
